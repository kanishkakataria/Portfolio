Predictive analysis of toolwear of milling Machine Using Acoustic Emissions

A solo project titled ‘Predictive Analysis of Tool Wear of Milling Machine using Acoustic Emission’.Spectrograms were created to predict tool wear through a functional model using Transfer Learning and Image Regression using algorithms like Cnn basic,Resnet50,InceptionV3,MobileNet

Indian Recipes Nutrition dataset creation and Nutritional Assessment System

A structured dataset is created based on Indian food recipes and given an overview of the whole process thus presenting the difficulties. The project includes crawling and scraping the data at first from the web by searching through 350+ different links.Specific things from the page is scraped and stored in an excel file which is then preprocessed and regression analysis is performed on the dataset to validate the data. The models applied on the data are CNN and LSTM. We compare the two modalities which are long text and image. As both the modalities show a significantly good performance, we have developed a nutrient assessment system that has a web interface to determine nutritional information based on the input received as an image of a dish or a recipe instruction.
